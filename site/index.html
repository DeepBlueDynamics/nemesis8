One-Shot & Interactive

run for clean output. interactive for full TUI. Detach and reattach anytime.

HTTP Gateway

Run prompts remotely. Schedule triggers. No local Docker needed on the client.

Resumable Sessions

Sessions persist across restarts. resume by ID. attach to running containers.

Pokeballs

Point at any GitHub repo or local project. nemesis8 figures out the language, installs dependencies, and seals it into a self-contained environment an AI can work in safely.

Getting Started

Up and Running in 60 Seconds.

1 Install

# paste in PowerShell
powershell -c "irm https://nemesis8.nuts.services/install.ps1 | iex"
# paste in Terminal
curl -fsSL https://nemesis8.nuts.services/install.sh | sh
# paste in terminal
curl -fsSL https://nemesis8.nuts.services/install.sh | sh

2 Build & Run

Pre-build the Docker image — optional, happens automatically on first run
nemesis8 build
One-shot prompt
nemesis8 run "list markdown files and summarize"
Interactive session
nemesis8 interactive
Switch providers
nemesis8 --provider gemini interactive
nemesis8 --provider claude interactive
nemesis8 --provider ala interactive
Remote mode (no local Docker needed)
nemesis8 --remote http://server:4000 run "hello"

3 Configure

# Initialize config in your project
nemesis8 init

# .nemesis8.toml
provider = "codex" # or "gemini"
workspace_mount_mode = "named"
mcp_tools = ["serpapi-search.py", "gnosis-crawl.py"]

[env]
MY_API_URL = "https://api.example.com"
env_imports = ["SERVICE_URL", "API_KEY"]

[[mounts]]
host = "C:/Users/you/data"
container = "/workspace/data"

4 Sessions & Resume

List past sessions
nemesis8 sessions
Resume by last 5 chars of UUID
nemesis8 resume 8d44d
Drop into a container shell
nemesis8 shell
Sealed Environments

The Pokeball System.

Point nemesis8 at any GitHub repo or local folder. It detects the language, installs the dependencies, and builds a self-contained image an AI can work inside — isolated from your machine.

Then run any prompt against it. The AI can read, write, and run code inside the sealed environment, but it can't reach the internet or your filesystem.

1
Seal a project
nemesis8 pokeball seal https://github.com/org/repo

Point at any GitHub repo or local folder. nemesis8 detects the language, installs dependencies, and builds a self-contained image.

2
Run a prompt
nemesis8 pokeball run repo --prompt "fix the failing tests"

The AI reads, writes, and runs code inside the sealed environment — it can't reach the internet or your filesystem. The original image is never modified.
